# 学生端移动应用说明

## 1. 模块定位

`Oline-AQ-app` 是面向学生手机端的 Android 原生应用，主要承载学生登录、考试作答、成绩复盘、在线练习、错题沉淀和个人资料维护等移动场景。

## 2. 当前功能

## 2.0 当前视觉风格

- 基于 `Material 3 + Catppuccin` 做高对比品牌化强化。
- 顶部标题区与主按钮统一使用蓝紫到青色的三段渐变。
- 核心卡片使用更明确的描边、阴影和分层色块，突出移动端阅读节奏。
- 底部导航采用更接近悬浮条的品牌面板样式，顶部栏标题统一居中。
- 顶部栏本身回收到更干净的中性色表面，并补齐状态栏安全区，避免标题贴顶。

### 2.1 底部导航

- 考试：查看考试列表并进入作答页。
- 成绩：查看成绩记录并进入成绩详情。
- 练习：按分类进行题库练习。
- 错题：查看全部错题与自建错题本。
- 我的：查看个人资料、系统公告、主题设置、服务器设置和退出登录。

### 2.2 考试功能

- 拉取学生可参加考试列表。
- 考试列表使用整行单卡布局展示单场试卷。
- 进入考试详情页作答。
- 考试详情页支持题号导航，可快速跳转到指定题目。
- 手动滚动题目时，题号导航会自动联动高亮当前题目。
- 支持一键跳到下一道未作答题，便于快速检查漏题。
- 展示倒计时与答题进度。
- 支持自动交卷。

### 2.3 成绩功能

- 成绩列表按试卷聚合展示，每张试卷只显示一条卡片。
- 点击试卷卡片进入历次作答列表页。
- 在历次作答页中可查看该试卷所有提交记录。
- 点击某一次提交进入成绩详情页。
- 在成绩详情页对题目提交纠错反馈。

### 2.4 练习功能

- 拉取题库和分类。
- 按分类过滤题目。
- 支持单选、判断、填空、简答练习。
- 作答过程中显示未作答/已填写状态提示。
- 本地判分并显示参考答案。

### 2.5 错题本功能

- 查看全部错题。
- 查看学生自建错题本。
- 新建、编辑、删除错题本。
- 将错题加入指定错题本。
- 从错题本详情中移除题目。
- 错题详情按考试分组卡片展示，并强化题目层级信息。
- 错题本与错题详情使用更强的分组卡片和描边题目卡片样式。
- 单选与判断类错题在详情页补充选项可视化展示，并突出正确答案。

### 2.6 个人中心功能

- 查看姓名、账号、邮箱、注册时间。
- 编辑个人资料。
- 修改密码。
- 查看系统公告并标记已读。
- 个人中心头部使用头像式信息卡，公告按卡片形式展示已读/未读状态。
- 个人中心和公告区域使用更高对比的品牌色块和层次化卡片。
- 公告空状态与普通信息卡统一图标风格和视觉语言。
- 切换主题模式与 AMOLED 纯黑模式。
- 配置服务器地址并退出登录。

## 3. 主要目录

- `app/src/main/java/com/onlineaq/student/ui/examlist/`：考试列表。
- `app/src/main/java/com/onlineaq/student/ui/examdetail/`：考试详情与作答。
- `app/src/main/java/com/onlineaq/student/ui/results/`：成绩列表。
- `app/src/main/java/com/onlineaq/student/ui/resultdetail/`：成绩详情与反馈。
- `app/src/main/java/com/onlineaq/student/ui/practice/`：在线练习。
- `app/src/main/java/com/onlineaq/student/ui/wrongbook/`：错题本与错题详情。
- `app/src/main/java/com/onlineaq/student/ui/profile/`：个人中心。
- `app/src/main/java/com/onlineaq/student/data/api/`：Retrofit 接口定义。
- `app/src/main/java/com/onlineaq/student/data/model/`：App 数据模型。

## 4. 构建方式

```sh
cd Oline-AQ-app
.\gradlew.bat :app:assembleDebug
```

构建成功后会生成调试包用于本地安装验证。
