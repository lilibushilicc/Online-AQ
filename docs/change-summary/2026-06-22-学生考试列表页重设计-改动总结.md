# 学生考试列表页重设计改动总结

## 1. 改动目标

根据提供的 HTML 设计稿，将学生端考试列表页从常规卡片列表升级为 Slate & Teal Executive 风格的玻璃拟态界面，与登录页设计语言统一。

## 2. 用户视角

- 页面顶部：标题"考试列表" + "学生作答中心"徽章 + 装饰发光背景
- 统计区：4 个玻璃卡片（当前可参加/已发布考试/历史提交/当前身份），带进度条和 teal 强调边框
- 搜索栏：圆角 pill 样式 + 筛选/排序按钮
- 考试卡片（2列网格）：玻璃拟态卡片，显示考试名/描述/状态徽章 + 时长/总分/重考三列 + 时间信息栏 + 历史提交标签 + 底部操作按钮
- 已结束考试：半透明 + 灰度效果，按钮禁用
- 底部：分页指示点

## 3. 前端实现

- **页面**：`StudentExams.vue` 完全重写
- **组件**：不再使用 `StatCards.vue`，改为内联玻璃拟态统计卡片
- **路由**：无变化
- **状态管理**：无变化（Pinia store）

### 主要改动

| 区域 | 改前 | 改后 |
|---|---|---|
| 统计卡片 | `StatCards` 组件（普通卡片） | 4 个玻璃拟态卡片 + 进度条 + teal 左边框装饰 |
| 搜索栏 | `el-input` 标准样式 | 自定义 pill 形搜索框 + 图标 |
| 考试卡片 | `el-card` 标准卡片 | 自定义玻璃拟态卡片（`glass-bg` + `backdrop-filter`） |
| 布局 | 全宽单列 | 2 列网格（桌面）→ 1 列（移动端） |
| 状态徽章 | `el-tag` | 自定义 `exam-status` 类（teal/amber/red/灰色） |
| 操作按钮 | `el-button` | 自定义 teal 渐变按钮 + 箭头图标 |
| 装饰元素 | 无 | 背景发光球体 + 底部指示点 |
