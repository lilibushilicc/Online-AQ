# P-13 P-14 P-16 功能优化改动总结

## 1. 改动目标

实现三个较小的体验优化：错题本筛选、考试前概览、删除确认框补齐。

## 2. 用户视角

- **P-13 错题本筛选**：在错题本详情页（含全部错题和个人错题本）顶部增加关键词搜索和题型下拉过滤，过滤后实时更新题目数量。
- **P-14 考试前概览**：进入考试页先看到概览卡（题目数量、考试时长、总分、考试时间），点击「开始作答」后才显示题目和倒计时。
- **P-16 删除确认补齐**：题库单题删除、文件删除、错题本移除单题、删除错题本四处在执行前弹出确认框。

## 3. 前端实现

### P-13 错题本筛选

- **文件**：`StudentWrongBookDetail.vue`
- 新增 `filterKeyword` 和 `filterType` 两个响应式变量
- 新增 `filteredGroups` 计算属性：对 `groups` 做客户端过滤（关键词匹配 `questionContent`，题型精确匹配）
- `totalQuestions` 改为基于 `filteredGroups` 计算
- 模板顶部添加 `<el-input>` 搜索框和 `<el-select>` 题型下拉
- `v-for` 由 `groups` 改为 `filteredGroups`

### P-14 考试前概览

- **文件**：`StudentExamDetail.vue`
- 新增 `started` ref，默认 `false`
- 新增 `startExam()` 函数：置 `started = true`，启动倒计时、注册滚动监听
- `onMounted` 中移除原来的 `startCountdown` 和 `scroll` 监听启动，改为仅在 `startExam` 中调用
- 模板包裹 `<template v-if="started">`，前面新增概览卡片显示题目数/时长/总分/时间
- 概览卡底部「开始作答」按钮触发 `startExam`

### P-16 删除确认补齐

- **`AdminQuestions.vue`**：
  - 单题删除：`el-button` 改为 `el-popconfirm` 包裹，确认后调用 `store.deleteQuestion`
  - 文件删除：`handleDeleteFile` 内加 `ElMessageBox.confirm` 弹窗
- **`StudentWrongBookDetail.vue`**：
  - 移出单题（`handleRemove`）：加 `ElMessageBox.confirm`
  - 删除错题本（`handleDeleteNotebook`）：加 `ElMessageBox.confirm`
  - 引入 `ElMessageBox`
