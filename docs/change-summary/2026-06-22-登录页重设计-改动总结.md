# 登录页重设计改动总结

## 1. 改动目标

根据提供的 HTML 设计稿（Slate & Teal Executive 风格），将原有登录页改为左右分栏品牌布局，提升视觉档次和用户体验。

## 2. 用户视角

- 打开页面后看到左侧品牌区（动态粒子背景 + 标语 + 功能特性）+ 右侧登录卡片
- 左侧：标签 "AI 驱动的考试生态系统"、标题 "全流程智能评分 重新定义 测评体验"、4 个功能特性项
- 右侧：玻璃卡片式登录表单（学号/邮箱 + 密码 + 忘记密码链接 + 登录按钮 + 注册链接 + 管理员入口按钮）
- 底部：隐私政策 / 服务条款 / 帮助中心链接
- 移动端：左侧品牌区隐藏，仅显示登录卡片

## 3. 前端实现

- **页面**：`LoginView.vue` 完全重写
- **组件**：沿用 `CanvasParticles.vue` 作为全屏动态粒子背景
- **路由**：无变化（`/login`）
- **状态管理**：无变化（Pinia store）
- **样式**：组件内 scoped CSS，清理 `style.css` 中的旧登录全局样式
