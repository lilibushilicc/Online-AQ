# 题库题目编辑改动总结

## 1. 改动目标
题库管理页面每道题缺少编辑入口，如需修改必须通过上传覆盖。本次为每行新增编辑按钮，点击弹出表单修改题目内容。

## 2. 用户视角
- 题库管理页每行操作列新增「编辑」按钮
- 点击弹出编辑对话框，可修改题干、题型、选项（仅单选）、正确答案、分值、分类
- 保存后自动刷新列表

## 3. 前端实现
- **API**：新增 `updateQuestionApi(questionId, data)`
- **AdminQuestions.vue**：
  - 新增 `editDialogVisible`、`editingQuestion`、`editForm` 响应式状态
  - `openEditDialog()` 将行数据填充到表单
  - `saveEdit()` 调用 `PUT /api/questions/{id}`，成功后刷新列表
  - 模板新增 `<el-dialog>` 编辑表单（按题型显示/隐藏选项字段）
  - 操作列宽度从 100px 增至 140px，增加编辑按钮

## 4. 后端实现
后端 `PUT /api/questions/{questionId}` 接口已存在，本次无需改动。

## 5. 数据库与数据流
数据流：前端编辑表单 → `PUT /api/questions/{id}` → `QuestionService.update()` → `questionMapper.updateById()`。

## 6. 接口说明

| 接口 | 方法 | 作用 | 主要参数 | 返回结果 |
|---|---|---|---|---|
| `/api/questions/{questionId}` | PUT | 更新题目 | `{ questionContent, questionType, correctAnswer, score, optionA~D, category }` | `Question` |

## 7. 关键文件

| 文件 | 作用 |
|---|---|
| `Oline-AQ-vue/src/api/index.ts` | 新增 `updateQuestionApi()` |
| `Oline-AQ-vue/src/views/admin/AdminQuestions.vue` | 新增编辑按钮、编辑弹窗和表单逻辑 |

## 8. 验证方式
```sh
cd Oline-AQ-vue && npx vue-tsc --noEmit
```
前端类型检查通过，无报错。

## 9. 风险与后续优化
- 当前不支持修改题目来源文件（sourceFileId），如需可后续扩展。
- 编辑时题型变更可能导致选项字段不符，后续可增加题型变更确认逻辑。
