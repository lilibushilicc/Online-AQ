# 在线练习权限修复改动总结

## 1. 改动目标

修复 Android 端学生无法打开「在线练习」页面的问题。学生点击底部导航栏「在线练习」后页面无响应，无法加载题目。

## 2. 用户视角

修复后学生可以正常打开在线练习页面，查看题目列表、选择分类筛选、作答并提交。

## 3. 后端实现

- **QuestionController**：将类级别的 `@AdminOnly("仅管理员可操作题库")` 注解移除，改为在需要管理员权限的方法上单独添加（create、update、delete、batch-delete、batch-score、batch-category、export）。
- 保留 `GET /api/questions` 和 `GET /api/questions/categories` 接口对学生开放。

## 4. 文件视角

| 文件 | 作用 |
|---|---|
| `Oline-AQ-spring/.../controller/QuestionController.java` | 将类级 `@AdminOnly` 下放到各管理员方法，开放 GET 查询接口 |

## 5. 风险与后续优化

- 当前所有已发布考试中的题目数据对学生可见，需确保题库中不包含敏感信息。
- 后续可按分类或权限粒度进一步细控题目可见范围。
