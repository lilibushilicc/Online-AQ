# 考试详情题号跳转改动总结

## 1. 改动目标

本次改动针对 `Oline-AQ-app` 的考试详情页，补充学生在手机端作答时快速跳转到指定题目的能力。

原实现只有顺序滚动浏览题目，题量多时学生需要不断上下滑动查找目标题目，尤其在回头检查漏题或修改答案时效率较低。为此，本次在考试详情页增加题号导航区，让学生可以点击题号直接跳到对应题目。

## 2. 用户视角

- 学生进入考试详情页后，可以在顶部信息区下方看到一排题号导航。
- 点击某个题号后，页面会自动滚动到对应题目。
- 当前跳转中的题号、已经作答的题号、未作答的题号有不同的视觉状态，便于快速定位。

## 3. 前端实现

这里的“前端”指 Android App 客户端界面实现。

- 页面：
  - 在 `activity_exam_detail.xml` 中增加题号导航区。
- 组件：
  - 使用 `HorizontalScrollView + LinearLayout` 作为题号容器。
  - 新增 `bg_question_nav_idle.xml`、`bg_question_nav_done.xml`、`bg_question_nav_active.xml` 三种状态背景。
- 交互：
  - `ExamDetailActivity.kt` 在题目加载完成后动态生成题号按钮。
  - 点击题号会调用 `RecyclerView.smoothScrollToPosition(index)` 滚动到目标题。
  - 已答题号、当前题号和未答题号根据答题状态更新样式。

## 4. 后端实现

- Controller：无改动。
- Service：无改动。
- Mapper：无改动。
- Entity / DTO / VO：无改动。
- 配置：无改动。

本次只涉及 Android 客户端考试详情页的交互增强。

## 5. 数据库与数据流

数据库无改动，数据流无改动。

题号导航使用的仍是考试详情接口返回的题目列表数据，只在客户端增加了导航视图和跳转行为。

## 6. 接口说明

| 接口 | 方法 | 作用 | 主要参数 | 返回结果 |
|---|---|---|---|---|
| `/api/exams/{examId}` | GET | 获取考试详情与题目列表 | `examId` | `ExamDetail` |

## 7. 关键文件

| 文件 | 作用 |
|---|---|
| `Oline-AQ-app/app/src/main/res/layout/activity_exam_detail.xml` | 新增题号导航区域 |
| `Oline-AQ-app/app/src/main/java/com/onlineaq/student/ui/examdetail/ExamDetailActivity.kt` | 动态生成题号并实现跳转逻辑 |
| `Oline-AQ-app/app/src/main/res/drawable/bg_question_nav_idle.xml` | 未作答题号样式 |
| `Oline-AQ-app/app/src/main/res/drawable/bg_question_nav_done.xml` | 已作答题号样式 |
| `Oline-AQ-app/app/src/main/res/drawable/bg_question_nav_active.xml` | 当前题号样式 |

## 8. 验证方式

```sh
cd Oline-AQ-app
.\gradlew.bat :app:assembleDebug
```

验证结果：Android 调试包构建成功。

## 9. 风险与后续优化

- 当前题号导航已支持快速跳转，但“当前题号”状态主要根据点击切换，后续可以继续接入滚动监听，让滚动浏览时也自动高亮当前可见题号。
- 后续还可以补充“仅跳转未作答题”或“定位下一道未作答题”等辅助能力。
