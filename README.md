## 项目简介

`Online-AQ` 是一个面向教学与练习场景的智能在线答题系统，提供从题目录入、组卷发布、学生作答、自动评分 / 人工评分，到成绩复盘、错题沉淀、题目反馈修正的完整闭环。

项目采用前后端分离架构，并同时提供：

- Web 管理端：面向教师 / 管理员，负责题库、考试、成绩、反馈、配置等后台管理。
- Web 学生端：面向学生，负责考试、成绩、练习、错题本、公告、个人中心等学习流程。
- Android 学生端 App：面向移动场景的原生学生客户端。

## 主要功能

### 管理员 / 教师端

- 上传 TXT / DOCX 文件并解析题目入库
- 题库管理：筛选、批量设分、批量分类、单题编辑、导出
- 创建考试、发布考试、设置开始 / 结束时间、乱序策略、重考策略
- 查看考试成绩、成绩详情、待评分简答题
- 处理学生题目纠错反馈
- 管理公告、用户、SMTP、AI 接口、对象存储等系统配置

### 学生端

- 查看可参加考试并在线作答
- 自动保存草稿并恢复作答
- 支持考试重考与历次提交记录查看
- 查看成绩详情、错题本、公告
- 在线练习与练习历史
- 新建 / 编辑 / 删除错题本
- 个人资料维护、密码修改、题目纠错反馈

### Android 学生端

- 覆盖考试、成绩、练习、错题本、个人中心等主流程
- 支持移动端考试详情题号导航、历史记录查看、公告查看等能力

## 快速启动

### 1. 数据库初始化

创建数据库：

```sql
CREATE DATABASE exam_system;
```

执行初始化脚本：

```text
init.sql
```

### 2. 启动后端

配置文件位置：

```text
Oline-AQ-spring/src/main/resources/application.yml
```

启动命令：

```sh
cd Oline-AQ-spring
.\mvnw.cmd spring-boot:run
```

默认地址：`http://localhost:8080`

### 3. 启动 Web 前端

```sh
cd Oline-AQ-vue
npm install
npm run dev
```

默认地址：`http://127.0.0.1:5173`

开发环境默认通过 `/api` 代理到后端。

### 4. 构建 Android 学生端

```sh
cd Oline-AQ-app
.\gradlew.bat :app:assembleDebug
```
